⚽ World Cup 2026 Calendar

All 104 matches with kickoff times, stadiums, results, and watch links.

Android (Google Calendar)

The Google Calendar app cannot import a shared .ics file. Use this page instead.

Add to Google Calendar Download calendar file

If the button does not work, open calendar.google.com in your phone browser (not the app), tap + next to Other calendars → From URL, and paste:

https://raw.githubusercontent.com/vcordero07/wc26-calendar/main/WC26-android.ics

The calendar syncs to your phone after a few minutes. For a one-time import instead, use Settings → Import & export → Import on calendar.google.com.
